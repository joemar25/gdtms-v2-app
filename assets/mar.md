## AppAssets constants (`lib/core/constants.dart`)

All asset paths are centralised in `AppAssets`. **Never hardcode `'assets/...'` strings in Dart files.**

| Constant                  | Path                                 | Used by                                                                                                               |
| ------------------------- | ------------------------------------ | --------------------------------------------------------------------------------------------------------------------- |
| `AppAssets.icon`          | `assets/android-chrome-512x512.png`  | `header.dart`, `splash_screen.dart`                                                                                   |
| `AppAssets.logo`          | `assets/logo.png`                    | Legacy fallback                                                                                                       |
| `AppAssets.animHourGlass` | `assets/anim/hour-glass.json`        | `loading_overlay.dart`, `splash_screen.dart`, `dashboard_screen.dart`, `initial_sync_screen.dart`, `sync_screen.dart` |
| `AppAssets.animSuccess`   | `assets/anim/successfully-done.json` | `success_overlay.dart`, `initial_sync_screen.dart`, `sync_screen.dart`                                                |
| `AppAssets.animEmpty`     | `assets/anim/empty.json`             | `empty_state.dart`                                                                                                    |
| `AppAssets.legalTerms`    | `assets/legal/terms.md`              | `terms_screen.dart`                                                                                                   |
| `AppAssets.legalPrivacy`  | `assets/legal/privacy.md`            | `privacy_screen.dart`                                                                                                 |

## Regenerating Launcher Icons & Splash

```bash
# Launcher icons (Android + iOS)
flutter pub run flutter_launcher_icons:main

# Native splash screen
# flutter pub run flutter_native_splash:create OR dart run flutter_native_splash:remove 2>&1 to remove the splash auto generated by flutter

# Clear build cache after asset changes
flutter clean && flutter pub get
```
